Project Overview

Offer buyers a new, proactive parcel collecting option with smart lockers.

This project aims to increase Shopee platform ADO by 6% by converting premium users who prefer to make high-value purchases of furniture and electrical devices offline.
The product covers:
- For buyers: New checkout flow with new channel preselection logic, map displaying, and other UI.
- For 3rd-party logistics partner: New API integration for orders with self-collection service.
